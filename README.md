# 사진대장 생성 Wep App

스마트폰 브라우저에서 앱 설치 없이 바로 쓸 수 있는 **사진대장(Photo Report) 작성 웹앱**입니다.
사진을 한 장씩(또는 여러 장) 추가하고 설명을 적으면, A4 규격 사진대장 문서를 자동으로 구성해 PDF로 출력하거나 메일로 보낼 수 있습니다.

🔗 **바로 사용하기**: https://audition4111-netizen.github.io/Photo-Report/

## 주요 기능

- 파일명 입력 후 PDF 출력
- 스마트폰 사진 앨범에서 사진 추가 (한 장 또는 여러 장 한 번에 선택, 최대 30장)
- 사진별 설명 입력 (여러 줄 지원, 자동 높이 조절, 출력 시 가운데 정렬)
- 사진 변경 / 90도 회전 / 삭제
- 손가락 드래그로 사진 순서 변경 (SortableJS)
- A4 세로 페이지에 사진 4장(2×2)씩 자동 배치, 5장부터 다음 페이지 자동 생성
- A4 미리보기 (실제 출력과 동일한 레이아웃)
- **PDF 출력**: 실제 PDF 파일 생성, 스마트폰에서 바로 다운로드
- **메일로 전송**: PDF를 자동 다운로드하면서 Gmail 작성 화면을 새 탭으로 열어줌 (첨부는 Gmail 화면에서 직접 추가)
- 서버/백엔드/데이터베이스/로그인 없음 — 모든 처리는 브라우저 안에서만 이루어지며, 새로고침하면 작업 내용은 사라짐

## 사용 방법

1. 파일명을 입력합니다.
2. [사진 추가]로 사진을 한 장 또는 여러 장 선택합니다.
3. 사진 아래에 설명을 입력합니다. (최대 30장까지 반복)
4. 필요하면 변경 / 회전 / 삭제, 드래그로 순서 변경을 합니다.
5. [미리보기]로 A4 페이지 구성을 확인합니다.
6. [PDF 출력]으로 다운로드하거나, [메일로 전송]으로 PDF 다운로드 + Gmail 작성 화면을 한 번에 엽니다.

####### 만든이의 사용 가이드 추천사항 #######

1. 물품 구매 후 ERP 첨부의 사진대장 작성 시 사용
 - 작성된 사진대장을 [메일로 전송]을 통해 회사메일로 전송
 - 자료 교환 후 ERP 첨부로 사용

2. 모바일 홈 화면에 배치하여 사용
 - 휴대폰 검색으로 [브라우저] 실행 후 **바로 사용하기의 주소를 복사하여 실행
 - 실행화면의 우측 하단 눌러 [현재 페이지 추가]-[홈 화면]으로 저장
