# 사진대장 생성 Wep App

스마트폰 브라우저에서 앱 설치 없이 바로 쓸 수 있는 **사진대장(Photo Report) 작성 웹앱**입니다.
사진을 한 장씩(또는 여러 장) 추가하고 설명을 적으면, A4 규격 사진대장 문서를 자동으로 구성해 PDF로 출력하거나 메일로 보낼 수 있습니다.

🔗 **바로 사용하기**: https://audition4111-netizen.github.io/Photo-Report/

## 주요 기능

- 파일명 입력 후 PDF 출력 (자동으로 `.pdf` 확장자 부여)
- 스마트폰 사진 앨범에서 사진 추가 (한 장 또는 여러 장 한 번에 선택, 최대 30장)
- 사진별 설명 입력 (여러 줄 지원, 자동 높이 조절, 출력 시 가운데 정렬)
- 사진 변경 / 90도 회전 / 삭제
- 손가락 드래그로 사진 순서 변경 (SortableJS)
- A4 세로 페이지에 사진 4장(2×2)씩 자동 배치, 5장부터 다음 페이지 자동 생성
- A4 미리보기 (실제 출력과 동일한 레이아웃)
- **PDF 출력**: html2canvas + jsPDF로 실제 PDF 파일 생성, 스마트폰에서 바로 다운로드
- **메일로 전송**: PDF를 자동 다운로드하면서 Gmail 작성 화면을 새 탭으로 열어줌 (첨부는 Gmail 화면에서 직접 추가)
- 서버/백엔드/데이터베이스/로그인 없음 — 모든 처리는 브라우저 안에서만 이루어지며, 새로고침하면 작업 내용은 사라짐

## 사용 방법

1. 파일명을 입력합니다.
2. [사진 추가]로 사진을 한 장 또는 여러 장 선택합니다.
3. 사진 아래에 설명을 입력합니다. (최대 30장까지 반복)
4. 필요하면 변경 / 회전 / 삭제, 드래그로 순서 변경을 합니다.
5. [미리보기]로 A4 페이지 구성을 확인합니다.
6. [PDF 출력]으로 다운로드하거나, [메일로 전송]으로 PDF 다운로드 + Gmail 작성 화면을 한 번에 엽니다.

## 기술 스택 / 사용 라이브러리

- HTML5 / CSS3 / Vanilla JavaScript (ES5, 별도 빌드 과정 없음)
- [SortableJS](https://github.com/SortableJS/Sortable) — 사진 순서 드래그 변경
- [html2canvas](https://github.com/niklasvh/html2canvas) — A4 페이지를 이미지로 캡처
- [jsPDF](https://github.com/parallax/jsPDF) — 캡처한 이미지를 PDF 문서로 조립

프레임워크나 번들러 없이 정적 파일만으로 동작하며, `libs/` 폴더에 위 3개 라이브러리를 로컬로 내려받아 포함하고 있어 **인터넷이 차단된 사내망에서도 통째로 복사해 오프라인으로 실행**할 수 있습니다.

## 폴더 구조

```
사진대장.html   # 원본 파일
index.html      # 사진대장.html과 동일 (GitHub Pages 루트 접속용)
libs/
  ├─ Sortable.min.js
  ├─ html2canvas.min.js
  └─ jspdf.umd.min.js
```

`사진대장.html`을 수정한 뒤에는 `index.html`도 동일하게 맞춰줘야 합니다(둘 다 같은 내용을 유지).

## 로컬 / 사내망 배포

1. 이 저장소를 통째로 다운로드하거나 복사합니다. (`libs/` 폴더 포함 필수)
2. 사내 웹서버에 올리거나, 공유 폴더에서 `index.html`을 더블클릭해 바로 엽니다.
3. 모든 라이브러리가 로컬 파일이라 외부 인터넷 연결 없이 동작합니다.

## 알려진 제한사항

- 브라우저 보안 정책상 어떤 웹사이트도 Gmail 등 외부 메일 서비스에 파일을 자동으로 첨부할 수 없습니다. [메일로 전송]은 PDF 다운로드 + Gmail 작성 화면 열기까지만 자동화하며, 첨부는 사용자가 직접 눌러야 합니다.
- 일부 구형 브라우저나 HEIC(아이폰 기본 사진 형식)를 지원하지 않는 환경에서는 사진 등록 시 안내 메시지가 표시됩니다.
