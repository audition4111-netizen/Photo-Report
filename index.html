이미지에서 여러 장을 한 번에 선택하거나, 스마트폰 기본 카메라 앱으로 바로 촬영해 등록할 수 있습니다.
사용 방법 안내 및 가이드
사용 순서
  1. 파일명을 입력합니다.
  2. [사진 추가]를 눌러 "이미지에서 선택"(앨범에서 한 장 또는 여러 장) 또는 "카메라로 촬영"(스마트폰 기본 카메라 앱으로 최고 화질 촬영) 중 선택합니다.
  3. 카메라로 촬영한 사진은 촬영 즉시 사진대장에 등록됩니다. 사진을 더 찍으려면 [사진 추가] → [카메라로 촬영]을 다시 눌러 주세요.
  4. 사진 아래 설명을 입력합니다.
  5. 필요한 만큼 사진을 반복해서 등록합니다. (최대 30장)
  6. 사진 변경 / 회전 / 삭제, 드래그로 순서 변경이 가능합니다.
  7. [미리보기]로 A4 페이지 구성을 확인합니다.
  8. [PDF 출력]으로 문서를 내려받거나, [메일로 전송]으로 PDF 다운로드와 Gmail 작성 화면 열기를 한 번에 실행합니다.
사용 가이드 추천
  1. 물품 구매 후 ERP 첨부의 사진대장 작성 시 사용
    • [메일로 전송]을 통해 회사메일로 전송
    • 자료 교환 후 ERP 첨부로 사용
  2. 모바일 홈 화면에 배치하여 사용
    • [브라우저]에서 실행
    • 우측 상단 또는 하단의 [현재 페이지 추가]-[홈 화면]
안내
  • 모든 사진/문서 처리는 이 브라우저 안에서만 이루어지며 외부 서버로 전송되지 않습니다.
  • [메일로 전송]을 누르면 PDF가 자동으로 다운로드되고 Gmail 작성 화면이 새 탭으로 열립니다. 첨부파일은 Gmail 화면에서 직접 추가해야 합니다(브라우저 보안 정책상 외부 웹사이트가 Gmail에 파일을 자동으로 넣을 수는 없습니다).
  • 새로고침하거나 화면을 닫으면 등록한 내용은 사라집니다.
