스마트폰 사진 앱에서 사진을 여러 장 한 번에 선택해 등록할 수 있습니다.
사용 방법 안내 / 라이브러리 정보
사용 순서
  1. 파일명을 입력합니다.
  2. [사진 추가]를 눌러 스마트폰 사진 앱에서 사진을 한 장 또는 여러 장 선택합니다.
  3. 사진 아래 설명을 입력합니다.
  4. 필요한 만큼 사진을 반복해서 등록합니다. (최대 30장)
  5. 사진 변경 / 회전 / 삭제, 드래그로 순서 변경이 가능합니다.
  6. [미리보기]로 A4 페이지 구성을 확인합니다.
  7. [PDF 출력] 또는 [DOCX 출력]으로 문서를 내려받습니다.
사용 라이브러리
  • SortableJS - 사진 순서 드래그 변경
  • html2canvas + jsPDF - A4 페이지를 이미지로 캡처해 PDF 생성
  • docx.js - 실제 편집 가능한 Word(.docx) 문서 생성
안내
  • 모든 사진/문서 처리는 이 브라우저 안에서만 이루어지며 외부 서버로 전송되지 않습니다.
  • 새로고침하거나 화면을 닫으면 등록한 내용은 사라집니다.
